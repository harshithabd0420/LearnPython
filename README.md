## Git Commands 
To Clone Repository(not needed in codespace) :
    git clone https URL

    git clone https https://github.com/harshithabd0420/LearnPython.git

To List the local branches:
    
    git branch

To Create a branch:

    git checkout -b pclass01

To See the latest local changes:

    git status

To check/verify the modified content in existing file:

    git diff

To stage the changes:

    git add <filename>

To Commit the changes:

    git commit -m "commit message"

To Push the changes:
    
    git push origin <sourceBranch>

            pclass01 -> main
            example: git push origin pclass01


### Daily

To check the branch is clean:

    git status

To checkout the main branch

    git checkout main

To get the latest changes

    git pull origin main

To create newbranch :
    
    git checkout -b <NEW BRANCH NAME>
